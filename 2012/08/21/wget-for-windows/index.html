Wget for Windows

2012-08-21

#wget

Not many things are this simple in life, but running Wget in Windows is one of them. First you will need to download wget.exe from the following address http://users.ugent.be/~bpuype/wget/ . Look for wget.exe and click on it to start the download. Once the executable has been downloaded you can either run it from the download directory. Move into the directory where wget.exe was downloaded to and execute the file.

cd Downlods

wget.exe

Or you can move the executable over to the C:Windows directory. This is my preferred method since it allows the user to call on wget without having to specify the path where the executable resides. But you will need administrators rights to move any file to C:Windows. Click on Continue. Time to use Wget, go to Start > Run > and type cmd. In the command window type wget. Congratulations you now have Wget in your Windows box.
