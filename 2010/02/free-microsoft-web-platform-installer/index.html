Free: Microsoft Web Platform Installer

The Microsoft Web Platform Installer 2.0 is a tool design help administrator install the latest components like Microsoft Web Platform, Internet Information Services (IIS), SQL Server Express, .NET Framework, PHP and Visual Web Developer just a few clicks away. Is not limited to just components you will also find the most popular free web applications for blogging (Wordpress), content management(Drupal) and more with the built-in Windows Web Application Gallery. No longer will you have to hunt down separate components, instead spend more time configuring rather than installing.
