Linhost site update

2009-03-02

Linhost.info will undergo a set of changes most of them aesthetic. The purpose of the change is to make the content easier to access and more organized. Special emphasis will be place on the virtual machine download area of this site. This is the result of all the feedback I received over the past few months.

  • Site presentation and layout will change
  • All available VMware virtual machines will include an md5sum
  • Virtual machines will be listed in alphabetical order
  • Unnecessary text will be removed from several pages
  • Some of the existing virtual machines will be updated with VMware Tools. This may take a long time to complete

The changes are for the best. Thank you for taking the time to leave feedback.
